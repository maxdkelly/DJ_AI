# DJ_AI

DJ_AI is a script that automatedly plays techno music.

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install the requirements for DJ_AI.

```bash
pip install -r requirements.txt
```

## Usage

```bash
python main.py
```
